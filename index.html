Política de Privacidade

ViaMark · Última atualização: 18 de junho de 2026

1. Introdução

O ViaMark é um aplicativo profissional para medição de sinalização horizontal em rodovias. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários.

2. Dados coletados

O ViaMark coleta e utiliza exclusivamente os seguintes dados:

  • Localização GPS — coletada durante o uso ativo do app para calcular a distância percorrida em cada elemento de sinalização medido. A localização também pode ser coletada em segundo plano quando uma medição está em andamento.

O app não coleta nome, e-mail, dados de contato, histórico de navegação, identificadores de dispositivo para fins publicitários, nem qualquer outra informação pessoal.

3. Uso dos dados

Os dados de localização são utilizados exclusivamente para:

  • Calcular a distância percorrida em tempo real durante uma medição;
  • Registrar as coordenadas de início e fim de cada trecho medido;
  • Gerar relatórios técnicos de sinalização horizontal.

4. Armazenamento

Todos os dados são armazenados localmente no dispositivo do usuário. O ViaMark não transmite dados de localização para servidores externos, não possui backend próprio e não compartilha dados com terceiros.

5. Compartilhamento de dados

O ViaMark não compartilha, vende ou transfere dados dos usuários a terceiros, sob nenhuma circunstância.

6. Permissões do sistema

O app solicita as seguintes permissões do iOS:

  • Localização quando em uso — necessária para medir sinalização com o app aberto na tela;
  • Localização sempre (em segundo plano) — necessária para continuar uma medição com o app minimizado.

O usuário pode revogar essas permissões a qualquer momento em Ajustes > ViaMark > Localização.

7. Retenção e exclusão

Os dados de medição ficam armazenados localmente até que o usuário os exclua manualmente dentro do app ou desinstale o aplicativo.

8. Segurança

Como os dados permanecem no dispositivo, estão protegidos pelas medidas de segurança nativas do iOS, incluindo criptografia do sistema de arquivos quando o dispositivo está bloqueado.

9. Menores de idade

O ViaMark é voltado a profissionais e não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

10. Alterações nesta política

Eventuais atualizações serão publicadas nesta página com a data de revisão atualizada. O uso continuado do app após alterações implica aceitação da política revisada.
